# Phalcon-MVC-mongodb

Made this to practice mongodb Operations

Services Used :-
Framework :- Phalcon
Architecture :- MVC
Phalcon Services :- 
  view, mongodb, url, namespace
Database :- Mongodb

Product Action :-
1. add new product
2. edit product
3. delete product
4. view all product
5. search product by name
6. add product variant and meta fields

Order Action :-
1. add new order
2. edit order
3. search order by status and placed date
